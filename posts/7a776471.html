Hexo 博客搭建教程
什么是 Hexo？Hexo 是一个基于 Node.js 的静态博客框架，支持快速生成静态网页并进行部署。它以简洁、高效和灵活著称，非常适合个人博客或小型网站。 安装 Hexo环境准备 安装 Node.js（建议使用稳定版）。 安装 Git。 安装 Hexo使用 npm 安装 Hexo： npm install -g hexo-cli 初始化博客新建一个博客文件夹（以英文命名），以文件夹路径运行以下命令： hexo init my-blog cd my-blog npm install 这样，你的博客框架就搭建完成了。 常用命令启动本地服务器 hexo server 访问 http://localhost:4000 查看你的博客。 清理缓存和静态文件hexo clean 生成静态文件hexo generate 部署博客hexo deploy 常见问题 部署失败？检查你的 _config.yml 文件是否正确配置 deploy 部分，确认仓库地址无误。 插件加载错误？确保所需的插件已通过 npm 安装，例如： npm install...
